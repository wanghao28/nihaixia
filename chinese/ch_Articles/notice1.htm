想來美國治病的病患須知

凡是要進入桃花島指名要找我看病的病人必須要注意以下數點，以免不受到我的歡迎。

第一 : 必須熟讀過我的網頁，沒有熟讀過的病患，我將不收治，一律交由我診所的醫師來治療。

第二 : 只知道跟我陳述西醫院說的病名，不知道要告訴我你自己身體的症狀者，也不歡迎。

第三 : 乳癌病患做過切片後再來我診所者，一律不被我接受，將交由我診所其他的醫師來治療。

第四 : 沒有做過切片與開刀化療的乳癌病患請注意，如果妳來自台灣，妳就必須負擔每次診療與藥費達65000元台幣，因為臺灣的健保局願意給付西藥廠的乳癌標靶藥物每劑65000元，一年十三次，而不願意給付中醫一毛錢，所以妳如果要抱怨，請妳去跟妳的健保局抱怨吧，跟我無關。(來自台灣以外地區的華人，不在此限。)

第五 : 如果你是腎臟癌患者，你進入桃花島的治療花費將達到每天3916元台幣，如果你嫌貴，請不要來這裡，取消你的機票吧，因為你來了還是白來，我不會幫你看病的，除非你願意付這種藥價。西藥廠 使用騙人的標靶藥物可以這樣收費，台灣政府居然還鼓勵，中醫藥同樣的收費，為何不被鼓勵呢？中西醫必須有公平的地位與尊嚴。(來自台灣以外地區的華人，不在此限。)

第六 : 你認為西醫比較科學，中醫不科學者，請不要來桃花島治病，我不收治像你這種笨蛋。

第七 : 你去西醫院做檢查後心生害怕，於是你很緊張的拿著檢驗報告來找中醫詢問，你必將被我趕出門去，因為那是西醫造成你很害怕，關我何事，又不是我讓你害怕的，請去詢問讓你害怕的西醫師吧。我看病可能只要花費一兩分鐘，做這種無謂的解釋，卻可能浪費我幾個小時，就算解釋完後，你還是照樣害怕，何苦呢。

第八 : 任何病人只要你講話讓我感覺到你是相信西醫的，請不要來此，相信西醫學就該繼續給西醫治療，不要來找碴。

第九 : 台灣來的病患，如果你從未到過我學生在台北的診所治病，請不要來此，我將不收治你。

第九 : 任何肝癌病人來診，如果你在得到肝癌之前曾經服用過抗肝炎病毒藥物，或是使用過干擾素來治療肝炎的話，請不要來桃花島治療，如果你硬要來此，也將由我的學生來診治，我不收治你這種肝癌。如果你因為服用西藥抗肝炎病毒藥而得到肝癌，也因此對西藥廠提出詐欺告訴，我就收治你，因為你已經覺醒，同時我要將你的性命保留下來，以便可以跟西藥廠求償 跟索命。

第十 : 如果你是來自台灣，但是卻騙我說你住在美國等，這將產生更壞的結果，我不幫專門騙人的人治病。我只要看你有沒有美國駕照就可以知道你是否騙我。

第十一 : 病患由我的學生推薦來的，將受到最好的接待，如果在外面自稱是我的學生，然後又推三阻四的不願意開推薦函給病人來找老師治病，這就是背叛師門不知尊師重道的劣徒，請病患也不用再找這種敗類學生治病了，這些學生只知利用老師的名聲來爭取病患，不知該自我進修醫術，這類心術不正者，必無法為良醫 。

第十二 : 如果你是大腸癌的病患，請注意你將支付的中藥價格，每次你來診時，就必須支付每次17985元台幣的藥費，如果不願意付，請不要來桃花島治病，大腸癌本來是不會致命的一種癌症，在中醫理論中是屬於腑病，而腑病本來就易治，大腸癌給西醫治療後百分之九十九會移轉成肝癌，成為臟病肝癌後就很難治了，甚至不治，所以給西醫治是越治越壞，在台灣使用無效的大腸直腸癌標靶藥物可以得到給付，為何有效的中醫治療就不能得到給付呢？(來自台灣以外地區的華人，不在此限。)

第十三 : 如果你的態度是要求中醫先將你的病治好，才會相信中醫，這種人就是我最痛恨的人，你信不信中醫學干我何事，這是你個人智商的問題，你也沒有如此重要到可以對中醫有舉足輕重的地位，這種人必將被我趕出門去，一律不收治，請你 回去繼續吃害死你的西藥吧。

第十四 : 任何病人先經過西醫開刀後再化療、放射療後，等到西醫宣佈治療無效後再來找我者，請不要再來桃花島，以免浪費你的金錢與寶貴的時間，當你有機會痊癒時，你卻將機會給予西醫院所，等到你只剩下奄奄一息一口氣時再來找中醫，此時你已經失去治病良機，請將錢財留給你的親人吧，本島不接待你這種病患。

所有病患在看診之前，必須先簽下此切結書才會獲准進入診間，本須知持續的更新中。不指名找我看病者，不在此限。

我ㄧ生幫人治病，鮮少有病人不滿意，只有我不幫人看病才會得罪人，目前已經得罪許多人了。所謂匹夫無罪，懷璧其罪。

美國漢唐中醫 桃花島島主倪藥師 敬上
